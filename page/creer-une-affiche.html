1) Télécharger le logiciel (libre et gratuit) GIMP. (si vous l'avez déjà c'est encore mieux !)
2) Télécharger le fichier "affiche-chagbo.xcf" puis exécutez-le avec GIMP
Télécharger GIMP
Télécharger "affiche-chagbo.xcf"
3) Vous aurez alors une interface comme celle-ci :



4) Vous pouvez donc modifier tout ce que vous souhaitez. (se référencer à la vidéo pour savoir comment procédé)
5) Une fois votre image terminée vous pouvez l'exporter. (se référencer à la vidéo pour savoir comment procédé)
6) Vous pouvez donc partager l'affiche ou vous souhaitez.
